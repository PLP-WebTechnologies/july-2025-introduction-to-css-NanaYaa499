The Box Model (Visual Demo)

Each element is a box: content inside padding, then border, then margin. The demo boxes below use the same declared width but different padding/border so you can observe the impact.

Demo A padding: 8px; border: 2px;
Demo B padding: 24px; border: 8px;

With box-sizing: content-box, padding and border increase total size. With box-sizing: border-box, total width stays fixed (padding/border are included).
